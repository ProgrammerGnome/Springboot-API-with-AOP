Kedves Szoftverfejlesztő!

Ennek a kódnak a funkcióit a következő linkek alatt tudja megtekinteni.

A tárhelyet a következő link alatt tudja elérni, melyhez "/ID"-t hozzáfűzve megtudja tekinteni a feltöltött elemeket egyenként is:
http://localhost:8080/storages     (Itt még sajnos FrontEnd nincs, csak JSON.)

Ezen az lapon pedig html oldal segítségével fel tudja tölteni a DB-be a PC-ről amit szeretne:
http://localhost:8080/upload

A következő műveletek léteznek még:

Kiválasztott fájl letöltése a DB-ből PC-re:    http://localhost:8080/download/ID
Kiválasztott fájl törlése a DB-ből:    http://localhost:8080/delete/ID

------------------ JDBC H2 felület ------------------
A link elvezeti önt a JDBC H2 konzolhoz: http://localhost:8080/h2-console/
A user name és password mezőket hagyja üresen. A JDBC URL pedig: jdbc:h2:~/springboot
